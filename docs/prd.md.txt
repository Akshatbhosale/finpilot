Product Requirements Document
Project: FinPilot
1. Purpose

FinPilot is a full-stack fintech web application that allows users to manage personal finances, track investments, and test investment strategies using historical market data.

The project demonstrates production-level software engineering including backend systems, frontend dashboards, security, and deployment infrastructure.

2. Goals

Build a real-world fintech platform

Demonstrate full-stack engineering

Implement secure authentication

Support investment strategy backtesting

Deploy using Docker and AWS

3. Target Users

Students learning finance

beginner investors

developers exploring trading strategies

4. Core Features (MVP)

Strategy Backtesting
